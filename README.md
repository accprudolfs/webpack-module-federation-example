# webpack-module-federation-example

## What it is?

webpack module federation example

## How to use

Install it and run in development mode:

```bash
npm i
npm start
```

Install it and build:

```bash
npm i
npm run build
```

Install it and bundle in production mode:

```bash
npm i
npm run bundle
```

Install it and bundle in analyze mode:

```bash
npm i
npm run analyze
```
